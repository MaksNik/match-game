How to play:

Memory is a counter game where the object is to find pairs.

When the game begins, all pictures are hidden.

To Play:

  1. Select a card skirt and a game difficulty.

  2. Press "Start" button.

  3. Select two cards to try to match the pictures.

  4. If you match the pictures you can go again.

  5. If they don't match it is the computer turn them.

  6. The player that finds all pairs wins!

  7. Have Fun!
